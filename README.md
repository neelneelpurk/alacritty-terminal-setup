# Alacritty Terminal Setup

## Installation Guide

### 1. Install Dependencies

Run the following command to install the required packages using Homebrew:

```bash
brew install alacritty tmux tmuxinator antidote starship font-jetbrains-mono-nerd-font
```

### 2. Copy Dotfiles

Copy all the files from the `dotfiles` directory to your home directory (`$HOME`):

```bash
cp -a dotfiles/. $HOME/
```
